# Production Workflow

## 1. Create Character

- Create or update the character notes in `06-Characters`.
- Keep each character visually consistent: species, colors, outfit, accessories, personality, and recurring props.

## 2. Generate Story

- Generate a Filmora-ready story prompt in `02-Stories`.
- Use [[03-Video-Configs/Default-Video-Config]] as the default format.
- The story note should contain only the Filmora prompt.
