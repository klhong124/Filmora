# Production Workflow

## 1. Idea

- Capture the premise in `00-Inbox`.
- Decide the emotion: wonder, sadness, fear, hope, regret, or awe.
- Choose the ending image before writing the full script.

## 2. Script

- Create a new note from [[Templates/Story-Video-Template]].
- Write the hook first.
- Draft the narration in short paragraphs for easy voice recording.
- Add visual prompts under each scene.

## 3. Voice

- Generate or record narration.
- Listen once without visuals to check rhythm and clarity.
- Remove repeated words, confusing names, and overly long sentences.

## 4. Visuals

- Generate a small set of keyframes before making the full video.
- Keep character design consistent across scenes.
- Save prompt notes and selected outputs in `05-Assets`.

## 5. Edit

- Build the timeline around narration first.
- Add visuals second.
- Add music and sound design last.
- Keep text on screen minimal.

## 6. Upload

- Export using [[03-Video-Configs/Default-Video-Config]].
- Create title, description, thumbnail, tags, and pinned comment.
- Record what worked and what to improve in the story note.

## Done Checklist

- [ ] Script complete
- [ ] Voice generated or recorded
- [ ] Visual prompts complete
- [ ] Images or clips generated
- [ ] Music selected
- [ ] Sound design added
- [ ] Thumbnail made
- [ ] Title and description ready
- [ ] Uploaded or scheduled
